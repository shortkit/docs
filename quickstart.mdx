Get your API keys, install the SDK, and display a video feed. By the end of this guide you will have a working vertical feed and your first piece of uploaded content.

## Before you begin

You need:
- A ShortKit account with API keys ([sign up](https://portal.shortkit.dev) or [contact us](https://calendly.com/michael-shortkit/20-min-meeting))
- An existing iOS application (SwiftUI or UIKit)

## Step 1: Get your API keys

<Steps>
  <Step title="Log in to the Admin Portal">
    Navigate to [portal.shortkit.dev](https://portal.shortkit.dev) and sign in with your organization credentials.
  </Step>
  <Step title="Navigate to API Keys">
    Go to **Settings → API Keys** in the sidebar.
  </Step>
  <Step title="Copy your keys">
    You will see two keys:
    - **Publishable key** (`pk_live_...`): Used in the client SDK. Safe to ship in your app binary.
    - **Secret key** (`sk_live_...`): Used for server-side API calls only.

    Copy the publishable key for SDK initialization, and the secret key for uploading content in Step 5.
  </Step>
</Steps>

<Warning>
  Never expose your secret key (`sk_live_...` or `sk_test_...`) in client-side code, source control, or your app binary. Use it only from your server or a secure CI environment.
</Warning>

## Step 2: Install the SDK

<Tabs>
  <Tab title="iOS (SPM)">
    In Xcode, go to **File → Add Package Dependencies** and enter the repository URL:

    ```
    https://github.com/AudienceKit/shortkit-ios
    ```

    Or add it to your `Package.swift`:

    ```swift Package.swift
    dependencies: [
        .package(url: "https://github.com/AudienceKit/shortkit-ios.git", from: "1.0.0")
    ]
    ```
  </Tab>
  <Tab title="Android">
    Add the dependency to your `build.gradle.kts`:

    ```kotlin build.gradle.kts
    dependencies {
        implementation("dev.shortkit:shortkit-android:1.0.0")
    }
    ```

    Add the ShortKit Maven repository in your `settings.gradle.kts`:

    ```kotlin settings.gradle.kts
    dependencyResolutionManagement {
        repositories {
            google()
            mavenCentral()
            maven { url = uri("https://maven.shortkit.dev") }
        }
    }
    ```
  </Tab>
  <Tab title="React Native">
    <CodeGroup>

    ```bash npm
    npm install @shortkit/react-native
    ```

    ```bash yarn
    yarn add @shortkit/react-native
    ```

    </CodeGroup>

    Then install iOS pods:

    ```bash
    cd ios && pod install
    ```
  </Tab>
</Tabs>

## Step 3: Initialize ShortKit

Create a `ShortKit` instance with your publishable key and a `FeedConfig`. Do this once, early in your app lifecycle.

<Tabs>
  <Tab title="SwiftUI">
    ```swift App.swift
    import SwiftUI
    import ShortKit

    @main
    struct MyApp: App {
        let shortKit = ShortKit(
            apiKey: "pk_live_your_publishable_key",
            config: FeedConfig(
                feedHeight: .fullscreen,
                controls: ControlsConfig(),
                theme: ThemeConfig()
            )
        )

        var body: some Scene {
            WindowGroup {
                ContentView(shortKit: shortKit)
            }
        }
    }
    ```
  </Tab>
  <Tab title="UIKit">
    ```swift AppDelegate.swift
    import UIKit
    import ShortKit

    @main
    class AppDelegate: UIResponder, UIApplicationDelegate {
        var shortKit: ShortKit!

        func application(
            _ application: UIApplication,
            didFinishLaunchingWithOptions launchOptions: [UIApplication.LaunchOptionsKey: Any]?
        ) -> Bool {

            shortKit = ShortKit(
                apiKey: "pk_live_your_publishable_key",
                config: FeedConfig(
                    feedHeight: .fullscreen,
                    controls: ControlsConfig(),
                    theme: ThemeConfig()
                )
            )

            return true
        }
    }
    ```
  </Tab>
</Tabs>

**Optional:** If you have logged-in users, set a user ID so ShortKit can personalize the feed:

```swift
shortKit.setUserId("user_12345")
```

Call `shortKit.clearUserId()` on logout.

## Step 4: Display the feed

<Tabs>
  <Tab title="SwiftUI">
    Drop `ShortKitFeedView` into any SwiftUI view. It handles its own layout and playback.

    ```swift ContentView.swift
    import SwiftUI
    import ShortKit

    struct ContentView: View {
        let shortKit: ShortKit

        var body: some View {
            ShortKitFeedView(shortKit: shortKit)
                .ignoresSafeArea()
        }
    }
    ```
  </Tab>
  <Tab title="UIKit">
    Present `ShortKitFeedViewController` or embed it as a child view controller.

    ```swift ViewController.swift
    import UIKit
    import ShortKit

    class ViewController: UIViewController {
        override func viewDidLoad() {
            super.viewDidLoad()

            let feedVC = ShortKitFeedViewController(shortKit: shortKit)
            addChild(feedVC)
            feedVC.view.frame = view.bounds
            feedVC.view.autoresizingMask = [.flexibleWidth, .flexibleHeight]
            view.addSubview(feedVC.view)
            feedVC.didMove(toParent: self)
        }
    }
    ```
  </Tab>
</Tabs>

Build and run. You will see an empty feed until you upload content in the next step.

## Step 5: Upload test content

Use the Upload API from your terminal to push a video into your feed. This requires your **secret key**.

<Steps>
  <Step title="Create an upload">
    ```bash
    curl -X POST https://api.shortkit.dev/v1/uploads \
      -H "X-API-Key: sk_live_your_secret_key" \
      -H "Content-Type: application/json" \
      -d '{
        "title": "My First Video",
        "description": "Testing ShortKit integration",
        "tags": ["test", "demo"],
        "customMetadata": {
          "articleUrl": "https://example.com/article"
        }
      }'
    ```

    Response:

    ```json
    {
      "data": {
        "id": "sk_content_abc123",
        "uploadUrl": "https://uploads.shortkit.dev/...",
        "status": "waiting_for_upload"
      }
    }
    ```
  </Step>
  <Step title="Upload the file to the signed URL">
    Use the `uploadUrl` from the response to PUT your video file:

    ```bash
    curl -X PUT "UPLOAD_URL_FROM_RESPONSE" \
      -H "Content-Type: video/mp4" \
      --data-binary @your-video.mp4
    ```

    Processing typically takes 1-2 minutes. Once complete, the video appears in your feed automatically.
  </Step>
  <Step title="Verify in your app">
    Relaunch your app (or wait for the feed to refresh). Your uploaded video should appear in the feed. Swipe up to scroll through content.
  </Step>
</Steps>

<Note>
  You can also upload content through the Admin Portal at [portal.shortkit.dev](https://portal.shortkit.dev) under **Content → Upload**.
</Note>
