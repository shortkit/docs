The Analytics API provides access to aggregated metrics for content performance, user engagement, and playback quality.

## Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| `GET` | `/v1/analytics/overview` | High-level metrics summary |
| `GET` | `/v1/analytics/content` | Content-level metrics |
| `GET` | `/v1/analytics/content/{id}` | Single content metrics |
| `GET` | `/v1/analytics/engagement` | Engagement metrics |
| `GET` | `/v1/analytics/quality` | Playback quality metrics |
| `GET` | `/v1/analytics/retention` | User retention data |

## Overview metrics

Get a high-level summary of key metrics.

```bash
GET /v1/analytics/overview
```

### Query parameters

<ParamField query="startDate" type="string" required>
  Start date (ISO 8601: `YYYY-MM-DD`).
</ParamField>

<ParamField query="endDate" type="string" required>
  End date (ISO 8601: `YYYY-MM-DD`).
</ParamField>

<ParamField query="granularity" type="string" default="day">
  Time granularity: `hour`, `day`, `week`.
</ParamField>

### Example request

```bash
curl "https://api.shortkit.dev/v1/analytics/overview?startDate=2024-01-01&endDate=2024-01-31" \
  -H "Authorization: Bearer sk_live_your_secret_key"
```

## Content analytics

Get metrics for all content or a specific item.

### List content metrics

```bash
GET /v1/analytics/content
```

<ParamField query="startDate" type="string" required>
  Start date.
</ParamField>

<ParamField query="endDate" type="string" required>
  End date.
</ParamField>

<ParamField query="limit" type="integer" default={20}>
  Items per page (1-100).
</ParamField>

<ParamField query="cursor" type="string">
  Pagination cursor.
</ParamField>

<ParamField query="sort" type="string" default="impressions">
  Sort field: `impressions`, `plays`, `completions`, `watchTime`.
</ParamField>

<ParamField query="order" type="string" default="desc">
  Sort order: `asc`, `desc`.
</ParamField>

<ParamField query="tags" type="string">
  Filter by tags (comma-separated).
</ParamField>

### Example request

```bash
curl "https://api.shortkit.dev/v1/analytics/content?startDate=2024-01-01&endDate=2024-01-31&sort=completions&limit=10" \
  -H "Authorization: Bearer sk_live_your_secret_key"
```

```bash
GET /v1/analytics/retention
```
