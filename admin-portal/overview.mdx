The shortkit Admin Portal is a web application for managing your video feed platform. It provides tools for content management, feed configuration, A/B testing, analytics, and ad management.

<Note>
  Access the Admin Portal at [dashboard.shortkit.dev](https://dashboard.shortkit.dev)
</Note>

## Dashboard overview

When you log in, you see the Overview dashboard with key metrics:

### Key metrics

- **Total views** (last 7 days)
- **Unique viewers**
- **Total watch time**
- **Average session duration**
- **Feed depth** (videos watched per session)

### Quick actions

- Upload content
- View recent uploads
- Check processing status
- View active experiments

### Health indicators

- SDK integration status
- API request volume
- Error rates
- Processing queue depth

## Quick start

### First-time setup

<Steps>
  <Step title="Get your API keys">
    Go to **Settings → API Keys** and copy your publishable key for SDK integration.
  </Step>
  <Step title="Configure your domain">
    Go to **Settings → Branding** and set up your branded CDN domain.
  </Step>
  <Step title="Set up webhooks">
    Go to **Settings → Webhooks** to receive notifications when content is ready.
  </Step>
  <Step title="Upload content">
    Go to **Content → Upload** and add your first videos.
  </Step>
  <Step title="Configure the feed">
    Go to **Feed → Configuration** to set up ranking weights and filters.
  </Step>
</Steps>

### Common tasks

| Task | Location |
|------|----------|
| Upload a video | Content → Upload |
| Check processing status | Content → Processing Queue |
| Pin content to top of feed | Content → Library → [Item] → Editorial |
| Adjust ranking weights | Feed → Ranking |
| Create an A/B test | Experiments → Create New |
| View watch time metrics | Analytics → Overview |
| Generate API keys | Settings → API Keys |
| Add team members | Settings → Team |

## Keyboard shortcuts

| Shortcut | Action |
|----------|--------|
| `⌘ K` / `Ctrl K` | Open command palette |
| `⌘ U` / `Ctrl U` | Quick upload |
| `⌘ /` / `Ctrl /` | Show shortcuts |
| `G then C` | Go to Content |
| `G then A` | Go to Analytics |
| `G then S` | Go to Settings |

## Mobile access

The Admin Portal is responsive and works on tablets. For quick tasks like checking analytics or approving content, mobile browsers are supported.

<Tip>
  For content upload and detailed configuration, we recommend using a desktop browser.
</Tip>
