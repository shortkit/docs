The built-in overlay templates (like `.news`) cover common layouts. When your product needs UI that differs from those templates -- a branded play button, a custom progress ring, a commerce panel -- disable the default overlay and build your own.

## Disable the built-in overlay

Set `videoOverlay: .none` in your `FeedConfig`. The SDK renders raw video with no UI layer on top.

    ```swift
    let feedVC = ShortKitFeedViewController(shortKit: shortKit)
    ```
  </Tab>
  <Tab title="Swift (SwiftUI)">

With `videoOverlay: .none`, your code is responsible for all playback controls, metadata display, and action handling.

## Subscribe to player publishers

`ShortKitPlayer` exposes Combine publishers that emit playback state in real time. Subscribe to these from your custom overlay to keep your UI in sync.

<Tabs>
  <Tab title="Swift">
    ```swift
    import Combine
    import ShortKitSDK

    let player = shortKit.player
    var cancellables = Set<AnyCancellable>()

    // Playback time -- fires at ~10 Hz while playing
    player.time
        .receive(on: DispatchQueue.main)
        .sink { time in
            // time.current  -- elapsed seconds (Double)
            // time.duration -- total duration (Double)
            // time.buffered -- buffered seconds (Double)
            let progress = time.duration > 0 ? time.current / time.duration : 0
            progressBar.setProgress(Float(progress), animated: true)
        }
        .store(in: &cancellables)

    // Player state transitions
    player.playerState
        .receive(on: DispatchQueue.main)
        .sink { state in
            switch state {
            case .playing:
                playPauseButton.setImage(UIImage(systemName: "pause.fill"), for: .normal)
            case .paused, .ready:
                playPauseButton.setImage(UIImage(systemName: "play.fill"), for: .normal)
            case .buffering:
                spinner.startAnimating()
            default:
                spinner.stopAnimating()
            }
        }
        .store(in: &cancellables)

    // Current content metadata
    player.currentItem
        .receive(on: DispatchQueue.main)
        .sink { item in
            titleLabel.text = item?.title
            authorLabel.text = item?.author
        }
        .store(in: &cancellables)

    // Mute state
    player.isMuted
        .receive(on: DispatchQueue.main)
        .sink { muted in
            let icon = muted ? "speaker.slash.fill" : "speaker.wave.2.fill"
            muteButton.setImage(UIImage(systemName: icon), for: .normal)
        }
        .store(in: &cancellables)
    ```
  </Tab>
  <Tab title="Kotlin">
    ```kotlin
    val player = shortKit.player

    // Playback time
    lifecycleScope.launch {
        player.time.collect { time ->
            val progress = if (time.duration > 0) time.current / time.duration else 0.0
            progressBar.progress = (progress * 100).toInt()

        player.playerState.collect { state ->
            when (state) {
                PlayerState.Playing -> playPauseButton.setIconResource(R.drawable.ic_pause)

                PlayerState.Buffering -> spinner.visibility = View.VISIBLE
                else -> spinner.visibility = View.GONE
            }
        }
    }

    // Current content metadata
    lifecycleScope.launch {
        player.currentItem.collect { item ->
            titleLabel.text = item?.title
            authorLabel.text = item?.author
        }
    }

    // Mute state
    lifecycleScope.launch {
        player.isMuted.collect { muted ->
            val icon = if (muted) R.drawable.ic_mute else R.drawable.ic_unmute
            muteButton.setIconResource(icon)
        }
    }
    ```

### Available publishers

| Publisher | Type | Description |
|---|---|---|
| `time` | `PlayerTime` | Elapsed time, total duration, and buffered duration. ~10 Hz while playing. |
| `playerState` | `PlayerState` | State transitions: `.idle`, `.loading`, `.ready`, `.playing`, `.paused`, `.seeking`, `.buffering`, `.ended`, `.error(String)`. |
| `currentItem` | `ContentItem?` | Metadata for the active content item. `nil` when nothing is loaded. |
| `isMuted` | `Bool` | Current mute state. |
| `playbackRate` | `Float` | Current playback speed (1.0, 1.5, 2.0). |
| `captionsEnabled` | `Bool` | Whether captions are currently on. |
| `activeCaptionTrack` | `CaptionTrack?` | The selected caption track. |
| `activeCue` | `VTTCue?` | The active caption cue at the current playback time. |
| `didLoop` | `LoopEvent` | Fires each time the video loops. Contains `contentId` and `loopCount`. |
| `feedTransition` | `FeedTransitionEvent` | Start and end of feed navigation. Contains `phase`, `from`, `to`, `direction`. |

Point-in-time reads are available without subscribing via `currentItemValue` and `playbackRateValue`.

## Issue player commands

Call methods on `ShortKitPlayer` to control playback from your overlay's gesture handlers and buttons.

<Tabs>
  <Tab title="Swift">
    ```swift
    let player = shortKit.player

    // Play / pause
    player.play()
    player.pause()

    // Seek to 15 seconds without auto-resuming
    player.seek(to: 15.0)

    // Seek and immediately resume playback
    player.seekAndPlay(to: 15.0)

    // Mute / unmute
    player.setMuted(true)
    player.setMuted(false)

    // Navigate the feed
    player.skipToNext()
    player.skipToPrevious()

    // Playback speed
    player.setPlaybackRate(2.0)   // 2x speed
    player.setPlaybackRate(1.0)   // restore normal speed

    // Captions
    player.setCaptionsEnabled(true)
    player.selectCaptionTrack(language: "en")

    // Content signals (powers recommendation engine)
    player.sendContentSignal(.positive)
    player.sendContentSignal(.negative)

    // Seek thumbnail for preview UI
    if let thumbnail = player.seekThumbnail(at: 30.0) {
        previewImageView.image = thumbnail
    }
    ```
  </Tab>
  <Tab title="Kotlin">
    ```kotlin
    val player = shortKit.player

    player.play()
    player.pause()
    player.seek(15.0)
    player.seekAndPlay(15.0)
    player.setMuted(true)
    player.skipToNext()
    player.skipToPrevious()
    player.setPlaybackRate(2.0f)
    player.setCaptionsEnabled(true)
    player.selectCaptionTrack("en")
    player.sendContentSignal(ContentSignal.Positive)
    ```
  </Tab>
  <Tab title="React Native">
    ```tsx

      const { play, pause, seek, setMuted, skipToNext } = useShortKitPlayer();

      return (
        <>
          <Button onPress={play} title="Play" />
          <Button onPress={pause} title="Pause" />
          <Button onPress={() => seek(15.0)} title="Seek to 0:15" />
          <Button onPress={() => setMuted(false)} title="Unmute" />
          <Button onPress={skipToNext} title="Next" />
        </>
      );
    }
    ```

### Command reference

| Method | Description |
|---|---|
| `play()` | Resume playback. |
| `pause()` | Pause playback. |
| `seek(to:)` | Seek to a time in seconds. Does not auto-resume. |
| `seekAndPlay(to:)` | Seek to a time in seconds and immediately resume. |
| `setMuted(_:)` | Set mute state. |
| `skipToNext()` | Navigate to the next feed item. |
| `skipToPrevious()` | Navigate to the previous feed item. |
| `setPlaybackRate(_:)` | Set playback speed (e.g. 1.0, 1.5, 2.0). |
| `sendContentSignal(_:)` | Send a `.positive` or `.negative` signal for the recommendation engine. |
| `setMaxBitrate(_:)` | Cap video bitrate in bits per second. Pass 0 to uncap. |
| `setCaptionsEnabled(_:)` | Enable or disable caption rendering. |
| `selectCaptionTrack(language:)` | Select a caption track by BCP 47 language code. |
| `seekThumbnail(at:)` | Returns a storyboard thumbnail `UIImage` at the given time, or `nil` if unavailable. |

## Full example: custom overlay with play/pause and progress bar

This self-contained `UIView` subclass shows a centered play/pause button and a bottom-edge progress bar. It subscribes to `ShortKitPlayer` publishers for state and issues commands from a tap gesture.

<Tabs>
  <Tab title="Swift">
    ```swift
    import Combine
    import ShortKitSDK
    import UIKit

    final class MinimalOverlayView: UIView {

        // MARK: - Dependencies

        private let player: ShortKitPlayer
        private var cancellables = Set<AnyCancellable>()

        // MARK: - UI

        private let playPauseButton: UIButton = {
            let button = UIButton(type: .system)
            button.translatesAutoresizingMaskIntoConstraints = false
            let config = UIImage.SymbolConfiguration(pointSize: 40, weight: .medium)
            button.setImage(UIImage(systemName: "play.fill", withConfiguration: config), for: .normal)
            button.tintColor = .white

            // Circular blur background
            let blur = UIVisualEffectView(effect: UIBlurEffect(style: .systemUltraThinMaterialDark))
            blur.isUserInteractionEnabled = false
            blur.translatesAutoresizingMaskIntoConstraints = false
            blur.layer.cornerRadius = 36
            blur.clipsToBounds = true
            button.insertSubview(blur, at: 0)
            NSLayoutConstraint.activate([
                blur.widthAnchor.constraint(equalToConstant: 72),
                blur.heightAnchor.constraint(equalToConstant: 72),
                blur.centerXAnchor.constraint(equalTo: button.centerXAnchor),
                blur.centerYAnchor.constraint(equalTo: button.centerYAnchor),
            ])
            return button
        }()

        private let progressBar: UIProgressView = {
            let bar = UIProgressView(progressViewStyle: .default)
            bar.translatesAutoresizingMaskIntoConstraints = false
            bar.progressTintColor = .white
            bar.trackTintColor = UIColor.white.withAlphaComponent(0.3)
            bar.progress = 0
            return bar
        }()

        // MARK: - Init

        init(player: ShortKitPlayer) {
            self.player = player
            super.init(frame: .zero)
            setupUI()
            bindPublishers()
        }

        required init?(coder: NSCoder) {
            fatalError("init(coder:) has not been implemented")
        }

        // MARK: - Layout

        private func setupUI() {
            addSubview(playPauseButton)
            addSubview(progressBar)

            NSLayoutConstraint.activate([
                playPauseButton.centerXAnchor.constraint(equalTo: centerXAnchor),
                playPauseButton.centerYAnchor.constraint(equalTo: centerYAnchor),
                playPauseButton.widthAnchor.constraint(equalToConstant: 72),
                playPauseButton.heightAnchor.constraint(equalToConstant: 72),

                progressBar.leadingAnchor.constraint(equalTo: leadingAnchor),
                progressBar.trailingAnchor.constraint(equalTo: trailingAnchor),
                progressBar.bottomAnchor.constraint(equalTo: bottomAnchor),
                progressBar.heightAnchor.constraint(equalToConstant: 2),
            ])

            playPauseButton.addTarget(self, action: #selector(playPauseTapped), for: .touchUpInside)
        }

        // MARK: - Publishers

        private func bindPublishers() {
            player.time
                .receive(on: DispatchQueue.main)
                .sink { [weak self] time in
                    guard time.duration > 0 else { return }
                    let fraction = Float(time.current / time.duration)
                    self?.progressBar.setProgress(fraction, animated: true)
                }
                .store(in: &cancellables)

            player.playerState
                .receive(on: DispatchQueue.main)
                .sink { [weak self] state in
                    guard let self else { return }
                    let config = UIImage.SymbolConfiguration(pointSize: 40, weight: .medium)
                    switch state {
                    case .playing:
                        self.playPauseButton.setImage(
                            UIImage(systemName: "pause.fill", withConfiguration: config),
                            for: .normal
                        )
                        UIView.animate(withDuration: 0.3) {
                            self.playPauseButton.alpha = 0
                        }
                    case .paused, .ready:
                        self.playPauseButton.setImage(
                            UIImage(systemName: "play.fill", withConfiguration: config),
                            for: .normal
                        )
                        self.playPauseButton.alpha = 1
                    case .buffering:
                        self.playPauseButton.alpha = 0
                    default:
                        break
                    }
                }
                .store(in: &cancellables)
        }

        // MARK: - Actions

        @objc private func playPauseTapped() {
            player.playerState
                .first()
                .sink { [weak self] state in
                    if state == .playing {
                        self?.player.pause()
                    } else {
                        self?.player.play()
                    }
                }
                .store(in: &cancellables)
        }
    }
    ```

    ```kotlin
    class MinimalOverlayView @JvmOverloads constructor(
        context: Context,
        private val player: ShortKitPlayer,
    ) : FrameLayout(context) {

        private val playPauseButton: ImageButton
        private val progressBar: ProgressBar

        init {
            inflate(context, R.layout.minimal_overlay, this)
            playPauseButton = findViewById(R.id.playPauseButton)
            progressBar = findViewById(R.id.progressBar)

            playPauseButton.setOnClickListener {
                if (player.playerStateValue == PlayerState.Playing) {
                    player.pause()
                } else {
                    player.play()
                }
            }
        }

        fun bind(lifecycleScope: LifecycleCoroutineScope) {
            lifecycleScope.launch {
                player.time.collect { time ->
                    if (time.duration > 0) {
                        progressBar.progress = ((time.current / time.duration) * 100).toInt()
                    }
                }
            }
            lifecycleScope.launch {
                player.playerState.collect { state ->
                    when (state) {
                        PlayerState.Playing -> {
                            playPauseButton.setImageResource(R.drawable.ic_pause)
                            playPauseButton.animate().alpha(0f).setDuration(300).start()
                        }
                        PlayerState.Paused -> {
                            playPauseButton.setImageResource(R.drawable.ic_play)
                            playPauseButton.alpha = 1f
                        }
                        else -> {}
                    }
                }
            }
        }
    }
    ```

## Architecture pattern

Custom overlays follow the same unidirectional data flow as the built-in templates:

1. **State flows down** -- subscribe to `ShortKitPlayer` publishers (`time`, `playerState`, `currentItem`, `isMuted`, etc.) and update your UI in each sink.
2. **Commands flow up** -- call `ShortKitPlayer` methods (`play()`, `pause()`, `seek(to:)`, `setMuted(_:)`, etc.) from gesture handlers and button actions.
3. **Semantic actions flow out** -- use `sendContentSignal()` for app-level actions like share, article navigation, and likes.

All publisher subscriptions should dispatch to the main queue via `.receive(on: DispatchQueue.main)` and be stored in a `Set<AnyCancellable>` that is cleaned up when your overlay is deallocated.

## Reference implementation

The SDK's built-in `NewsOverlayView` (~1,000 lines of UIKit code) is a production-grade overlay that uses every publisher and command documented above. It includes a scrubber with seek preview thumbnails, expandable text, caption display, double-tap like animation, and long-press 2x speed. If you are building a complex overlay, use it as an architectural reference for how to structure bindings, manage gesture recognizer priority, and handle feed transition lifecycle events.
