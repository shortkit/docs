            February 2026

            Feb 11, 2026

            Mux Video now supports updating track details

            We’ve added support to update track details for an existing asset track via API. You can update the language, name, passthrough, and closed-captions information for any existing track.

            Feb 9, 2026

            Fixed timezone conversion issue in usage dashboard and CSV exports

            Fixed bug where usage dashboard converted UTC to local timezone. Data now clearly labeled as UTC for accurate reporting.

            Feb 3, 2026

            Mux Video now accepts assets with HDR10 as non-standard input

            Mux Video now accepts assets with HDR10 metadata non-standard input. Mux will process the asset to SDR BT.709 video. This also includes HDR10+ assets.

            December 2025

            Dec 18, 2025

            Usage Reports Now Available in Mux Dashboard

            New Usage Reports shows detailed breakdowns of your Mux Video usage across encoding, storage, and delivery over time.

            Dec 16, 2025

            Mux has released an open source toolkit for integrating Mux Video and AI providers

            Mux has released @mux/ai, an open-source TypeScript SDK for building AI-powered video features with Mux, including summarization, moderation, translation, and more.

            November 2025

            Nov 26, 2025

            Mux Data now supports Enhanced Rendition Dimensions & Rendition Change Metrics

            Adds extended support for Rendition Change metric tracking with 9 new dimensions and 3 new metrics.

            Nov 10, 2025

            DRM (Digital Rights Management) now generally available

            Mux Video DRM is now generally available. With DRM enabled your video content will be encrypted and protected from most screen capture tools.

            Nov 7, 2025

            Mux Data SDKs now support Playback Mode Change Events

            Web and Mobile SDKs now support playback mode change events. Integrate playback mode change events to get access to Playing Time by Playback Mode in Streaming Exports V15+.

            Nov 7, 2025

            Mux Data SDKs now support Ad Type

            Web and Mobile SDKs support ad type metadata for ad events and are automatically calculated in the Google IMA SDK. Enables Playing Time by Ad Type in Streaming Exports V15+.

            October 2025

            Oct 27, 2025

            Sanity plugin for Mux Video now supports title search

            The Sanity plugin for Mux now includes automatic title search and a backfill tool for existing videos.

            Oct 15, 2025

            HEVC Standard Input Now Available to All Users

            HEVC standard input support is now available to all Mux users, enabling faster processing for mobile video uploads.

            Oct 10, 2025

            Custom Dashboards now support sharing

            You can now share Custom Dashboards you create on Mux Data. Shared dashboards provide viewer access to all members of an environment.

            Oct 9, 2025

            Mux Data now supports saving and sharing filter sets

            Save and share filter combinations in Mux Data. Filter sets can be shared or private. Create filter sets for future events.

            Oct 7, 2025

            Dashboard layout refresh now generally available

            The refreshed Mux dashboard layout and navigation are now generally available for all customers, completing the September rollout.

            Oct 7, 2025

            Historical bill reporting is now available

            Invoice history and spending analysis for the past six billing periods are now available to organization administrators.

            September 2025

            Sep 24, 2025

            Dashboard layout refresh now available for Data Media customers.

            The refreshed Mux dashboard layout, featuring simpler navigation and clearer org/env structure, is now available for Data Media and Pro plans.

            Sep 19, 2025

            Mux Video now supports HEVC as standard input

            Mux Video now supports HEVC encoded video as a ‘standard’ codec for VOD ingest. Both 8 bit and 10 bit HEVC are treated as standard.

            Sep 12, 2025

            It is now possible to create static rendition (MP4s) at specific resolutions

            It is now possible to create static MP4 renditions at specific resolutions: 270p, 360p, 480p, 540p, 720p, 1080p, 1440p, and 2160p for more precise resolution control.

            Sep 8, 2025

            Dashboard navigation and layout refresh

            The refreshed Mux dashboard is rolling out in phases with simpler navigation, a redesigned org/environment switcher, and streamlined top‑level pages for easier access.

            August 2025

            Aug 28, 2025

            Custom Dashboards: Favoriting and Export to PDF

            Dashboard favoriting and PDF export are now available for custom dashboards, enabling quick access to frequently used views and offline sharing capabilities.

            Aug 22, 2025

            Mux Remote MCP Release

            Mux's remote MCP server is now available, making it easy to empower your LLM with all of Mux's Video and Analytics capabilities: https://www.mux.com/docs/integrations/mcp-server.

            Aug 14, 2025

            Web and Mobile SDK support for CDN change events

            Web and Mobile SDKs now support CDN change events to track when CDN switches occur during video playback.

            Aug 14, 2025

            Mux Data launches CDN Trace dimension and enhanced multi-CDN tracking

            Track multi-CDN performance with Mux Data's new CDN Trace dimension. See which networks deliver your video content and when switches occur.

            July 2025

            Jul 25, 2025

            Static renditions can no longer be accessed with a DRM playback policy

            Static renditions can no longer be downloaded using playback IDs with DRM playback policies. Instead, you can use a signed playback policy.

            Jul 24, 2025

            Latest thumbnails for live streams

            It is now possible to request the latest thumbnail for an active live stream via the image.mux.com endpoint, using a new ?latest=true query parameter.

            Jul 21, 2025

            Mux Video Live Stream auto-generated closed captions now supports additional languages

            Mux Video Live auto-generated closed captions now supports additional languages: French, German, Italian, Portuguese, and Spanish.

            Jul 17, 2025

            Mux Video self-service pricing has been improved

            Mux Video's pricing has been improved. Encoding is reduced by 22%, storage by 23% for basic, and streaming by 17%. Cold storage support has been expanded. Static renditions are now a paid add-on.

            Jul 17, 2025

            Basic quality level assets now support 2K and 4K resolutions

            Basic quality assets can now be ingested at 2K (1440p) and 4K (2160p) resolutions while still benefitting from free encoding. Previously, basic quality level assets were limited to 1080p.

            Jul 14, 2025

            Updated User Agent for File Ingestion

            Mux will use `curl/7.43.0 (Mux)` as the user agent for file ingestion starting July 21st.

            Jul 8, 2025

            Custom Dashboards: Build persistent, multi-metric dashboard views in Mux Data

            Create persistent, multi-metric dashboard views in Mux Data. Save configurations with up to 10 components and flexible filtering options.

            June 2025

            Jun 2, 2025

            The domain of URLs returned for Direct Uploads are changing

            Starting April 10, 2025, Direct Upload API URLs will start to use a subdomain of mux.com instead of storage.googleapis.com. Update your domain allow-lists and CSP if needed.

            May 2025

            May 30, 2025

            More resolutions are now supported in the min_resolution and max_resolution playback modifiers

            The list of supported values for the min_resolution and max_resolution playback modifiers has been updated to include the full range of standard resolutions.

            May 16, 2025

            Mux Data Now Supports IPv6 Connections

            Mux Data now supports IPv6 connections, improving geolocation accuracy and supporting IPv6-only devices with no additional cost to customers.

            May 13, 2025

            The completion percentage of non-standard input processing is now available in the assets API

            The assets API now exposes the state and approximate completion percentage when processing a non-standard input.

            May 12, 2025

            Standardized case formatting for text_type in new_asset_settings

            The Direct Uploads and Live Streams APIs have been updated to return the text_type field within new_asset_settings in the same lower-case formatting as the request type.

            May 12, 2025

            The default filename and URL for a master download has been changed from "master" to "mezzanine"

            The default filename for a master download has been changed from master.mp4/m4a to mezzanine.mp4/m4a. These files are also now served from mezzanine.mux.com, which replaces master.mux.com.

            May 12, 2025

            It is now possible to customize the filename for a master download

            It is now possible to customize the filename used by a master download by setting the download query parameter on the master access URL.

            May 8, 2025

            Updated Sign In and Sign Up Experience

            We’ve updated our Sign In and Sign Up experiences. There is no change to existing functionality, but we think it’s pretty cool lookin’.

            May 1, 2025

            The Mux Video Starter Plan has been improved

            Accounts using the Mux Video Starter Plan now benefit from 100,000 minutes of free video delivery every month.

            May 1, 2025

            The Mux Video Pay as you go Plan has been improved

            Accounts using the Mux Video Pay as you go Plan now benefit from 100,000 minutes of free video delivery, and $20 of usage credit every month.

            May 1, 2025

            Mux Video now has a Free Plan

            The new Mux Video Free Plan lets accounts upload 10 assets, deliver 100K mins/month, and use features like Mux Player, engagement analytics, generated captions, and multi-track audio.

            April 2025

            Apr 29, 2025

            Mux Data now supports Annotations

            Mux Data supports Annotations allowing you to tag moments with a datetime, note, and a sub-property ID. You can create, update, delete and show annotations within the dashboard and via API.

            Apr 29, 2025

            New cursor pagination added to list assets endpoint

            The list asset API now supports cursor pagination. When paginating through very large lists of assets you will find that cursor pagination is faster and more reliable.

            Apr 24, 2025

            New Self-Service Media Plan for Mux Data

            You can now sign up for Mux Data Media which includes real-time monitoring, streaming exports, and more. Billed at $499/month and additional views at $0.0005 per view.

            Apr 10, 2025

            Title and additional metadata now supported on assets

            Mux now supports three metadata fields to help you organize and manage your video content across the API and dashboard: title, creator_id and external_id.

            March 2025

            Mar 27, 2025

            Long-Term Metrics: 13 Months of Video Analytics Now Generally Available

            Long-Term Metrics is now generally available, offering 13 months of video analytics retention for Custom Media plans through the Mux Dashboard and Data API.

            Mar 10, 2025

            Mux Data SDK for AVPlayer v4.4.0 Released

            Version 4.4.0 adds support for several new dimensions you can add to your views. Examples include dynamic range type, video and audio codec, CDN PoP, CDN Origin, and more.

            Mar 4, 2025

            Automatic cold storage improvements

            New assets are eligible for cold storage discounts immediately, until first played. Cold storage now has no performance penalty when playing assets.

            Mar 4, 2025

            Mux Player for Android v1.4.0

            v1.4.0 of Mux Player for Android improves its data on Mux Data. You can now update `CustomerData` after building your `MuxPlayer`, and the player will automatically track the resolution of the player

            February 2025

            Feb 20, 2025

            Additional Standard Dimensions

            Adds support for 16 new dimensions, all available to be set in the Data SDKs.
