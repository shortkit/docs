---
title: "Get Event"
api: "GET /v1/events/{id}"
description: "Retrieve a single event by ID."
---

Returns the full event record including its `data` payload.

## Path parameters

<ParamField path="id" type="string" required>
  Event UUID.
</ParamField>

<RequestExample>
```bash cURL
curl "https://api.shortkit.dev/v1/events/e8f9a0b1-c2d3-4e5f-6a7b-8c9d0e1f2a3b" \
  -H "Authorization: Bearer sk_live_your_secret_key"
```

```python Python
import requests

resp = requests.get(
    "https://api.shortkit.dev/v1/events/e8f9a0b1-c2d3-4e5f-6a7b-8c9d0e1f2a3b",
    headers={"Authorization": "Bearer sk_live_your_secret_key"},
)
event = resp.json()["data"]
```

```javascript Node.js
const resp = await fetch(
  "https://api.shortkit.dev/v1/events/e8f9a0b1-c2d3-4e5f-6a7b-8c9d0e1f2a3b",
  {
    headers: { Authorization: "Bearer sk_live_your_secret_key" },
  }
);
const { data: event } = await resp.json();
```
