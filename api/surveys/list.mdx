---
title: "List Surveys"
api: "GET /v1/surveys"
description: "Returns all surveys in your organization, ordered by priority."
---

## Query parameters

<ParamField query="status" type="string">
  Filter by status: `draft`, `active`, `paused`, or `archived`.
</ParamField>

<RequestExample>
```bash cURL
curl "https://api.shortkit.dev/v1/surveys?status=active" \
  -H "Authorization: Bearer sk_live_your_secret_key"
```

```python Python
import requests

resp = requests.get(
    "https://api.shortkit.dev/v1/surveys",
    headers={"Authorization": "Bearer sk_live_your_secret_key"},
    params={"status": "active"},
)
surveys = resp.json()["data"]
```

```javascript Node.js
const resp = await fetch(
  "https://api.shortkit.dev/v1/surveys?status=active",
  {
    headers: { Authorization: "Bearer sk_live_your_secret_key" },
  }
);
const { data: surveys } = await resp.json();
```
</RequestExample>
