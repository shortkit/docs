---
title: "Update Survey"
api: "PUT /v1/surveys/{id}"
description: "Update a survey's fields. Options and placement rules are replaced entirely when provided."
---

## Path parameters

<ParamField path="id" type="string" required>
  Survey UUID.
</ParamField>

## Request body

All fields are optional — include only the fields you want to change.

<ParamField body="title" type="string">
  Updated title (max 200 characters).
</ParamField>

<ParamField body="question" type="string">
  Updated question text (max 500 characters).
</ParamField>

<ParamField body="status" type="string">
  New status: `draft`, `active`, `paused`, or `archived`.
</ParamField>

<ParamField body="priority" type="integer">
  Updated priority value.
</ParamField>

<ParamField body="autoAdvanceDelay" type="number">
  Updated auto-advance delay in seconds.
</ParamField>

<ParamField body="options" type="object[]">
  Replaces all existing options. Each must include `text` and `position`.
</ParamField>

<ParamField body="placementRules" type="object[]">
  Replaces all existing placement rules. Each must include `ruleType` and `ruleValue`.
</ParamField>

<Warning>
  When you provide `options` or `placementRules`, the entire set is replaced — not merged. Omit these fields to leave them unchanged.
</Warning>
