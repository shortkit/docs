---
title: "Get Survey"
api: "GET /v1/surveys/{id}"
description: "Retrieve a single survey by ID, including options, placement rules, and response counts."
---

## Path parameters

<ParamField path="id" type="string" required>
  Survey UUID.
</ParamField>

<RequestExample>
```bash cURL
curl "https://api.shortkit.dev/v1/surveys/d4e5f6a7-b8c9-4d0e-1f2a-3b4c5d6e7f8a" \
  -H "Authorization: Bearer sk_live_your_secret_key"
```

```python Python
import requests

resp = requests.get(
    "https://api.shortkit.dev/v1/surveys/d4e5f6a7-b8c9-4d0e-1f2a-3b4c5d6e7f8a",
    headers={"Authorization": "Bearer sk_live_your_secret_key"},
)
survey = resp.json()["data"]
```

```javascript Node.js
const resp = await fetch(
  "https://api.shortkit.dev/v1/surveys/d4e5f6a7-b8c9-4d0e-1f2a-3b4c5d6e7f8a",
  {
    headers: { Authorization: "Bearer sk_live_your_secret_key" },
  }
);
const { data: survey } = await resp.json();
```
</RequestExample>
