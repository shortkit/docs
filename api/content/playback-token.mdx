---
title: "Create Playback Token"
api: "POST /v1/content/{contentId}/playback-token"
description: "Issue a short-lived signed streaming URL for a protected content item."
---

Issue a **signed streaming URL** for a protected content item. Protected content withholds its streaming URL until a signed URL is issued — this endpoint returns one, along with a matching signed thumbnail URL. Both URLs are short-lived and expire at the returned `expiresAt` time.

This endpoint accepts a **publishable key**, so it's safe to call directly from mobile apps and browsers.

<Note>
  You only need this endpoint when you play protected content **outside** the ShortKit SDK — for example, in a custom web player. The SDK issues and refreshes signed URLs automatically. See [Protected Content](/guides/protected-content).
</Note>

## Path parameters

<ParamField path="contentId" type="string" required>
  The content item's UUID.
</ParamField>

## Request body

Both fields are optional — an empty body (`{}`) is valid.

<ParamField body="thumbnailTime" type="integer" default="0">
  Seconds into the video for the signed thumbnail URL. Must be `0` or greater.
</ParamField>

<ParamField body="client" type="string">
  The app bundle ID or web origin making the request. Checked against your organization's optional signed-URL allowlist. When the allowlist is configured and `client` is not permitted, the request returns `403`.
</ParamField>

<RequestExample>
```bash cURL
curl -X POST "https://api.shortkit.dev/v1/content/d290f1ee-6c54-4b01-90e6-d701748f0851/playback-token" \
  -H "X-API-Key: pk_live_your_publishable_key" \
  -H "Content-Type: application/json" \
  -d '{
    "thumbnailTime": 5,
    "client": "com.example.app"
  }'
```

```python Python
import requests

resp = requests.post(
    "https://api.shortkit.dev/v1/content/d290f1ee-6c54-4b01-90e6-d701748f0851/playback-token",
    headers={
        "X-API-Key": "pk_live_your_publishable_key",
        "Content-Type": "application/json",
    },
    json={
        "thumbnailTime": 5,
        "client": "com.example.app",
    },
)
if resp.status_code == 200:
    token = resp.json()
    video_url = token["video"]["url"]
    expires_at = token["video"]["expiresAt"]
else:
    error = resp.json()["errors"][0]
    print(f"Playback token failed: {error['code']} — {error['message']}")
```

## Response

<ResponseField name="playbackId" type="string">
  The content item's playback ID.
</ResponseField>

<ResponseField name="video" type="object">
  The signed streaming URL for playback.
  - `url` — HLS streaming URL (`.m3u8`) with an embedded signed token.
  - `expiresAt` — Unix epoch seconds after which the URL stops working.
</ResponseField>

<ResponseField name="thumbnail" type="object">
  The signed thumbnail URL.
  - `url` — Thumbnail image URL with an embedded signed token, taken at `thumbnailTime`.
  - `expiresAt` — Unix epoch seconds after which the URL stops working.
</ResponseField>

<Note>
  The signed URLs are short-lived. Request a fresh token before `expiresAt` to keep playback uninterrupted.
</Note>

## Errors

| Status | Description |
|--------|-------------|
| **403** | The `client` is not permitted by your organization's signed-URL allowlist. |
| **404** | Content not found, or it doesn't belong to your organization. |
| **409** | The content item is not protected. Use its public `streamingUrl` from the [content object](/api/content/object) instead. |
| **422** | The content's media asset is not ready. Wait until `uploadStatus` is `ready`, then retry. |
| **500** | Signed playback is not configured. |
