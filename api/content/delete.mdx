---
title: "Delete Content"
api: "DELETE /v1/content/{id}"
description: "Permanently delete a content item. The item is removed from the feed and list results."
---

## Path parameters

<ParamField path="id" type="string" required>
  The content item's UUID.
</ParamField>

Deleting a content item permanently removes it — the content is immediately dropped from the feed and no longer appears in list, search, or count results. The underlying media assets are cleaned up asynchronously.

This operation is not reversible.

<RequestExample>
```bash cURL
curl -X DELETE "https://api.shortkit.dev/v1/content/d290f1ee-6c54-4b01-90e6-d701748f0851" \
  -H "Authorization: Bearer sk_live_your_secret_key"
```

```python Python
import requests

resp = requests.delete(
    "https://api.shortkit.dev/v1/content/d290f1ee-6c54-4b01-90e6-d701748f0851",
    headers={"Authorization": "Bearer sk_live_your_secret_key"},
)
```

```javascript Node.js
const resp = await fetch(
```
