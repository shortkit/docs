---
title: "Get Content"
api: "GET /v1/content/{id}"
description: "Retrieve a single content item by ID."
---

## Path parameters

<ParamField path="id" type="string" required>
  The content item's UUID.
</ParamField>

<RequestExample>
```bash cURL
curl "https://api.shortkit.dev/v1/content/d290f1ee-6c54-4b01-90e6-d701748f0851" \
  -H "Authorization: Bearer sk_live_your_secret_key"
```

```python Python
import requests

resp = requests.get(
    "https://api.shortkit.dev/v1/content/d290f1ee-6c54-4b01-90e6-d701748f0851",
    headers={"Authorization": "Bearer sk_live_your_secret_key"},
)
content = resp.json()["data"]
```

```javascript Node.js
const resp = await fetch(
  "https://api.shortkit.dev/v1/content/d290f1ee-6c54-4b01-90e6-d701748f0851",
  {
    headers: { Authorization: "Bearer sk_live_your_secret_key" },
  }
);
const { data: content } = await resp.json();
```
</RequestExample>
