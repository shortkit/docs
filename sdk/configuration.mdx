Every configuration field has a sensible default. `FeedConfig()` with no arguments produces a working feed.

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `apiKey` | `String` | **required** | Your publishable API key. Starts with `pk_live_` or `pk_test_`. |
| `config` | `FeedConfig` | **required** | Feed layout and behavior configuration. See [FeedConfig](#feedconfig) below. |
| `userId` | `String?` | `nil` | Known user ID. When provided, the SDK starts in identified mode immediately, skipping the anonymous-first flow. |
| `environment` | `Environment` | `.production` | API environment. Controls endpoint routing. See [Environment](#environment). |
| `adProvider` | `ShortKitAdProvider?` | `nil` | Ad provider for monetization. Pass `nil` to disable ads. See [Ad integration](/sdk/ads). |
| `clientAppName` | `String?` | `nil` | App name included in analytics metadata. |
| `clientAppVersion` | `String?` | `nil` | App version included in analytics metadata. |
| `customDimensions` | `[String: String]?` | `nil` | Custom key-value pairs for analytics segmentation. Keys must be `"custom_1"` through `"custom_5"`. |

On initialization, the SDK pre-fetches the first feed page, pre-warms the CDN connection, and begins HLS preroll for the first video. The feed view controller can render immediately without a loading spinner.

## `FeedConfig`

<Tabs>
  <Tab title="Swift">
    ```swift
    public struct FeedConfig: Codable {
        public var feedHeight: FeedHeight
        public var videoOverlay: VideoOverlayMode
        public var carouselOverlay: CarouselOverlayMode
        public var surveyOverlay: SurveyOverlayMode
        public var adOverlay: AdOverlayMode
        public var muteOnStart: Bool

        public init(
            feedHeight: FeedHeight = .fullscreen,
            videoOverlay: VideoOverlayMode = .none,
            carouselOverlay: CarouselOverlayMode = .template(.default),
            surveyOverlay: SurveyOverlayMode = .template(.default),
            adOverlay: AdOverlayMode = .template(.default),
            muteOnStart: Bool = true
        )
    }
    ```
  </Tab>
  <Tab title="Kotlin">
    ```kotlin
    data class FeedConfig(
        val feedHeight: FeedHeight = FeedHeight.Fullscreen,
        val videoOverlay: VideoOverlayMode = VideoOverlayMode.None,
        val carouselOverlay: CarouselOverlayMode = CarouselOverlayMode.Template(CarouselTemplate.Default),
        val surveyOverlay: SurveyOverlayMode = SurveyOverlayMode.Template(SurveyTemplate.Default),
        val adOverlay: AdOverlayMode = AdOverlayMode.Template(AdOverlayTemplate.Default),
        val muteOnStart: Boolean = true
    )
    ```
  </Tab>
  <Tab title="React Native">
    ```typescript
    interface FeedConfig {
      feedHeight?: 'fullscreen' | { percentage: number };
      videoOverlay?: 'none' | { template: string } | { custom: ComponentType };
      carouselOverlay?: 'none' | { template: 'default' } | { custom: ComponentType };
      surveyOverlay?: 'none' | { template: 'default' } | { custom: ComponentType };
      adOverlay?: 'none' | { template: 'default' } | { custom: ComponentType };
      muteOnStart?: boolean;
    }
    ```
  </Tab>
</Tabs>

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `feedHeight` | `FeedHeight` | `.fullscreen` | `.fullscreen` fills the entire screen. `.percentage(CGFloat)` sets a fraction of screen height (e.g. `0.85` for 85%). |
| `videoOverlay` | `VideoOverlayMode` | `.none` | `.none` disables built-in overlays. `.template(...)` renders a pre-built overlay with action icons, progress bar, and content metadata. `.custom { ... }` provides a developer-built overlay managed by the SDK. See [Overlays](/sdk/overlays). |
| `carouselOverlay` | `CarouselOverlayMode` | `.template(.default)` | `.none` disables built-in carousel UI. `.template(.default)` renders horizontal paged images with page dots, caption, and auto-scroll. `.custom { ... }` provides a developer-built carousel overlay. See [Overlays](/sdk/overlays#carousel-survey-and-ad-overlay-modes). |
| `surveyOverlay` | `SurveyOverlayMode` | `.template(.default)` | `.none` disables built-in survey UI. `.template(.default)` renders a styled survey with question label, option buttons, and auto-advance after selection. `.custom { ... }` provides a developer-built survey overlay. See [Overlays](/sdk/overlays#carousel-survey-and-ad-overlay-modes). Responses are delivered via `ShortKitDelegate.shortKit(_:didRespondToSurvey:with:)`. |
| `adOverlay` | `AdOverlayMode` | `.template(.default)` | `.none` disables built-in native ad overlays. `.template(.default)` renders the "Ad" badge, AdChoices icon, sponsor logo, headline, and CTA button. `.custom { ... }` provides a developer-built ad overlay. See [Overlays](/sdk/overlays#carousel-survey-and-ad-overlay-modes). Only applies to native ads; system-defined ads (IMA/VAST) are unaffected. |
| `muteOnStart` | `Bool` | `true` | Whether video starts muted. Once the user unmutes, the unmuted state persists for the session. |

When using a built-in overlay template, set `overlayActionDelegate` on the `ShortKit` instance to handle user taps on overlay buttons. See [Overlays](/sdk/overlays) for details.

## `Environment`

<Tabs>
  <Tab title="Swift">

    ```swift
        case development
        case custom(URL)
    }
    ```
  </Tab>
  <Tab title="Kotlin">
    ```kotlin
    sealed class Environment {
        object Production : Environment()

        object Development : Environment()
        data class Custom(val url: String) : Environment()
    }
    ```
  </Tab>
  <Tab title="React Native">
    ```typescript
    type Environment = 'production' | 'staging' | 'development' | { custom: string };
    ```
  </Tab>
</Tabs>

| Case | Base URL |
|------|----------|
| `.production` | `https://api.shortkit.dev` |
| `.staging` | `https://api.staging.shortkit.dev` |
| `.development` | `https://api.dev.shortkit.dev` |
| `.custom(URL)` | Any URL you provide. Useful for local development or proxy setups. |

## Complete example

A full initialization with all configuration options set explicitly.

<Tabs>
  <Tab title="Swift (SwiftUI)">
    ```swift App.swift
    import SwiftUI
    import ShortKit

    @main
    struct MyApp: App {
        let shortKit: ShortKit

        init() {
            let config = FeedConfig(
                feedHeight: .fullscreen,
                videoOverlay: .none,
                carouselOverlay: .template(.default),
                surveyOverlay: .template(.default),
                adOverlay: .template(.default),
                muteOnStart: true
            )

            shortKit = ShortKit(
                apiKey: "pk_live_your_publishable_key",
                config: config,
                userId: nil,
                environment: .production,
                clientAppName: "MyNewsApp",
                clientAppVersion: "2.4.1",
                customDimensions: ["custom_1": "sports"]
            )

            shortKit.delegate = self
            shortKit.overlayActionDelegate = self
        }

        var body: some Scene {
            WindowGroup {
                ShortKitFeedView(shortKit: shortKit)
                    .ignoresSafeArea()
            }
        }
    }
    ```
  </Tab>
  <Tab title="Swift (UIKit)">
    ```swift ViewController.swift
    import UIKit
    import ShortKit

    class FeedViewController: UIViewController {
        private var shortKit: ShortKit!

        override func viewDidLoad() {
            super.viewDidLoad()

            let config = FeedConfig(
                feedHeight: .fullscreen,
                videoOverlay: .none,
                carouselOverlay: .template(.default),
                surveyOverlay: .template(.default),
                adOverlay: .template(.default),
                muteOnStart: true
            )

            shortKit = ShortKit(
                apiKey: "pk_live_your_publishable_key",
                config: config,
                userId: nil,
                environment: .production,
                clientAppName: "MyNewsApp",
                clientAppVersion: "2.4.1",
                customDimensions: ["custom_1": "sports"]
            )

            shortKit.delegate = self
            shortKit.overlayActionDelegate = self

            let feedVC = ShortKitFeedViewController(shortKit: shortKit)
            addChild(feedVC)
            feedVC.view.frame = view.bounds
            feedVC.view.autoresizingMask = [.flexibleWidth, .flexibleHeight]
            view.addSubview(feedVC.view)
            feedVC.didMove(toParent: self)
        }
    }
    ```

    ```kotlin MainActivity.kt
    import android.os.Bundle
    import androidx.appcompat.app.AppCompatActivity
    import dev.shortkit.ShortKit
    import dev.shortkit.FeedConfig
    import dev.shortkit.FeedHeight
    import dev.shortkit.VideoOverlayMode
    import dev.shortkit.OverlayTemplate
    import dev.shortkit.CarouselOverlayMode
    import dev.shortkit.CarouselTemplate
    import dev.shortkit.SurveyOverlayMode
    import dev.shortkit.SurveyTemplate
    import dev.shortkit.AdOverlayMode
    import dev.shortkit.AdOverlayTemplate
    import dev.shortkit.Environment
    import dev.shortkit.ShortKitFeedView

    class MainActivity : AppCompatActivity() {
        override fun onCreate(savedInstanceState: Bundle?) {
            super.onCreate(savedInstanceState)

            val config = FeedConfig(
                feedHeight = FeedHeight.Fullscreen,
                videoOverlay = VideoOverlayMode.None,
                carouselOverlay = CarouselOverlayMode.Template(CarouselTemplate.Default),
                surveyOverlay = SurveyOverlayMode.Template(SurveyTemplate.Default),
                adOverlay = AdOverlayMode.Template(AdOverlayTemplate.Default),
                muteOnStart = true
            )

            val shortKit = ShortKit(
                apiKey = "pk_live_your_publishable_key",
                config = config,
                userId = null,
                environment = Environment.Production,
                clientAppName = "MyNewsApp",
                clientAppVersion = "2.4.1",
                customDimensions = mapOf("custom_1" to "sports")
            )

            val feedView = ShortKitFeedView(this, shortKit)
            setContentView(feedView)
        }
    }
    ```
