Every configuration field has a sensible default. `FeedConfig()` with no arguments produces a working feed.

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `apiKey` | `String` | **required** | Your publishable API key. Starts with `pk_live_` or `pk_test_`. |
| `config` | `FeedConfig` | **required** | Feed layout and behavior configuration. See [FeedConfig](#feedconfig) below. |
| `userId` | `String?` | `nil` | Known user ID. When provided, the SDK starts in identified mode immediately, skipping the anonymous-first flow. |
| `environment` | `Environment` | `.production` | API environment. Controls endpoint routing. See [Environment](#environment). |
| `adProvider` | `ShortKitAdProvider?` | `nil` | Ad provider for monetization. Pass `nil` to disable ads. See [Ad integration](/sdk/ads). |
| `clientAppName` | `String?` | `nil` | App name included in analytics metadata. |
| `clientAppVersion` | `String?` | `nil` | App version included in analytics metadata. |
| `customDimensions` | `[String: String]?` | `nil` | Custom key-value pairs for analytics segmentation. Keys must be `"custom_1"` through `"custom_5"`. |

On initialization, the SDK pre-fetches the first feed page, pre-warms the CDN connection, and begins HLS preroll for the first video. The feed view controller can render immediately without a loading spinner.

## `FeedConfig`

<Tabs>
  <Tab title="Swift">
    ```swift
    public struct FeedConfig: Codable, Sendable {
        public var feedHeight: FeedHeight
        public var overlay: OverlayMode
        public var carouselMode: CarouselMode
        public var surveyMode: SurveyMode
        public var muteOnStart: Bool

        public init(
            feedHeight: FeedHeight = .fullscreen,
            overlay: OverlayMode = .none,
            carouselMode: CarouselMode = .template(.default),
            surveyMode: SurveyMode = .template(.default),
            muteOnStart: Bool = true
        )
    }
    ```
  </Tab>
  <Tab title="Kotlin">
    ```kotlin
    data class FeedConfig(
        val feedHeight: FeedHeight = FeedHeight.Fullscreen,
        val overlay: OverlayMode = OverlayMode.None,
        val carouselMode: CarouselMode = CarouselMode.Template(CarouselTemplate.Default),
        val surveyMode: SurveyMode = SurveyMode.Template(SurveyTemplate.Default),
        val muteOnStart: Boolean = true
    )
    ```
  </Tab>
  <Tab title="React Native">
    ```typescript
    interface FeedConfig {
      feedHeight?: 'fullscreen' | { percentage: number };
      overlay?: 'none' | { template: string } | { custom: ComponentType };
      carouselMode?: 'none' | { template: 'default' } | { custom: ComponentType };
      surveyMode?: 'none' | { template: 'default' } | { custom: ComponentType };
      muteOnStart?: boolean;
    }
    ```
  </Tab>
</Tabs>

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `feedHeight` | `FeedHeight` | `.fullscreen` | `.fullscreen` fills the entire screen. `.percentage(CGFloat)` sets a fraction of screen height (e.g. `0.85` for 85%). |
| `overlay` | `OverlayMode` | `.none` | `.none` disables built-in overlays. `.template(...)` renders a pre-built overlay with action icons, progress bar, and content metadata. `.custom { ... }` provides a developer-built overlay managed by the SDK. See [Overlays](/sdk/overlays). |
| `carouselMode` | `CarouselMode` | `.template(.default)` | `.none` disables built-in carousel UI. `.template(.default)` renders horizontal paged images with page dots, caption, and auto-scroll. `.custom { ... }` provides a developer-built carousel overlay. See [Overlays](/sdk/overlays#carousel-and-survey-modes). |
| `surveyMode` | `SurveyMode` | `.template(.default)` | `.none` disables built-in survey UI. `.template(.default)` renders a styled survey with question label, option buttons, and auto-advance after selection. `.custom { ... }` provides a developer-built survey overlay. See [Overlays](/sdk/overlays#carousel-and-survey-modes). Responses are delivered via `ShortKitDelegate.shortKit(_:didRespondToSurvey:with:)`. |
| `muteOnStart` | `Bool` | `true` | Whether video starts muted. Once the user unmutes, the unmuted state persists for the session. |

When using a built-in overlay template, set `overlayActionDelegate` on the `ShortKit` instance to handle user taps on overlay buttons. See [Overlays](/sdk/overlays) for details.

## `Environment`

<Tabs>
  <Tab title="Swift">

    ```swift
        case development
        case custom(URL)
    }
    ```
  </Tab>
  <Tab title="Kotlin">
    ```kotlin

        data class Custom(val url: String) : Environment()
    }
    ```
  </Tab>
  <Tab title="React Native">
    ```typescript
    type Environment = 'production' | 'staging' | 'development' | { custom: string };
    ```
  </Tab>
</Tabs>

| Case | Base URL |
|------|----------|
| `.production` | `https://api.shortkit.dev` |
| `.staging` | `https://api.staging.shortkit.dev` |
| `.development` | `https://api.dev.shortkit.dev` |
| `.custom(URL)` | Any URL you provide. Useful for local development or proxy setups. |

## Complete example

A full initialization with all configuration options set explicitly.

<Tabs>
  <Tab title="Swift (SwiftUI)">
    ```swift App.swift
    import SwiftUI
    import ShortKit

    @main
    struct MyApp: App {
        let shortKit: ShortKit

        init() {
            let config = FeedConfig(
                feedHeight: .fullscreen,
                overlay: .none,
                carouselMode: .template(.default),
                surveyMode: .template(.default),
                muteOnStart: true
            )

            shortKit = ShortKit(
                apiKey: "pk_live_your_publishable_key",
                config: config,
                userId: nil,
                environment: .production,
                clientAppName: "MyNewsApp",
                clientAppVersion: "2.4.1",
                customDimensions: ["custom_1": "sports"]
            )

            shortKit.delegate = self
            shortKit.overlayActionDelegate = self
        }

        var body: some Scene {
            WindowGroup {
                ShortKitFeedView(shortKit: shortKit)
                    .ignoresSafeArea()
            }
        }
    }
    ```
  </Tab>

    ```swift ViewController.swift
    import UIKit

    class FeedViewController: UIViewController {
        private var shortKit: ShortKit!

        override func viewDidLoad() {
            super.viewDidLoad()

            let config = FeedConfig(
                feedHeight: .fullscreen,
                overlay: .none,
                carouselMode: .template(.default),
                surveyMode: .template(.default),
                muteOnStart: true

            shortKit = ShortKit(
                apiKey: "pk_live_your_publishable_key",
                config: config,
                userId: nil,
                environment: .production,
                clientAppName: "MyNewsApp",
                clientAppVersion: "2.4.1",
                customDimensions: ["custom_1": "sports"]
            )

            shortKit.delegate = self
            shortKit.overlayActionDelegate = self

            let feedVC = ShortKitFeedViewController(shortKit: shortKit)
            addChild(feedVC)
            feedVC.view.frame = view.bounds
            feedVC.view.autoresizingMask = [.flexibleWidth, .flexibleHeight]
            view.addSubview(feedVC.view)
            feedVC.didMove(toParent: self)
        }
    }
    ```

    ```kotlin MainActivity.kt
    import android.os.Bundle
    import androidx.appcompat.app.AppCompatActivity
    import dev.shortkit.ShortKit
    import dev.shortkit.FeedConfig
    import dev.shortkit.FeedHeight
    import dev.shortkit.OverlayMode
    import dev.shortkit.OverlayTemplate
    import dev.shortkit.CarouselMode
    import dev.shortkit.CarouselTemplate
    import dev.shortkit.SurveyMode
    import dev.shortkit.SurveyTemplate
    import dev.shortkit.Environment
    import dev.shortkit.ShortKitFeedView

    class MainActivity : AppCompatActivity() {
        override fun onCreate(savedInstanceState: Bundle?) {
            super.onCreate(savedInstanceState)

            val config = FeedConfig(
                feedHeight = FeedHeight.Fullscreen,
                overlay = OverlayMode.None,
                carouselMode = CarouselMode.Template(CarouselTemplate.Default),
                surveyMode = SurveyMode.Template(SurveyTemplate.Default),
                muteOnStart = true
            )

            val shortKit = ShortKit(
                apiKey = "pk_live_your_publishable_key",
                config = config,
                userId = null,
                environment = Environment.Production,
                clientAppName = "MyNewsApp",
                clientAppVersion = "2.4.1",
                customDimensions = mapOf("custom_1" to "sports")
            )

            val feedView = ShortKitFeedView(this, shortKit)
            setContentView(feedView)
        }
    }
    ```
