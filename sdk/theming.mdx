The ShortKit SDK provides theming options to match your brand's visual identity. All visual properties are configured through the `ThemeConfig` struct, which is set on `FeedConfig.theme`. Every field has a sensible default, so you only need to customize what matters for your brand.

## ThemeConfig

<ResponseField name="primaryColor" type="Color" default=".white">
  Accent color for interactive elements such as the scrubber fill and active toggle states. Pass any valid platform color.
</ResponseField>

<ResponseField name="backgroundColor" type="Color" default=".black">
  Feed background color. Visible during transitions between videos or if content does not fill the viewport.
</ResponseField>

<ResponseField name="controlsTint" type="Color" default=".white">
  Tint color for all control icons (play/pause, share, captions, volume, etc.).
</ResponseField>

<ResponseField name="controlsBackgroundOpacity" type="CGFloat" default="0.4">
  Opacity of the gradient overlay behind controls (0.0 to 1.0). Ensures control visibility over video content.
</ResponseField>

<ResponseField name="fontFamily" type="String" default="System">
  Font for all text rendered by the SDK, including captions and labels. Falls back to the system font if the specified font is unavailable.
</ResponseField>

<ResponseField name="scrubberStyle" type="ScrubberStyle" default="ScrubberStyle()">
  Detailed styling for the progress bar and scrubber. See `ScrubberStyle` reference below.
</ResponseField>

<ResponseField name="captionStyle" type="CaptionStyle" default="CaptionStyle()">
  Detailed styling for caption rendering. See `CaptionStyle` reference below.
</ResponseField>

### Basic example

<Tabs>
  <Tab title="SwiftUI">
    ```swift
    let shortKit = ShortKit(
        apiKey: "pk_live_your_publishable_key",
        config: FeedConfig(
            theme: ThemeConfig(
                primaryColor: .orange,
                backgroundColor: .black,
                controlsTint: .white,
                controlsBackgroundOpacity: 0.5,
                fontFamily: "System"
            )
        )
    )

    ShortKitFeedView(shortKit: shortKit)
    ```
  </Tab>

## Scrubber styling

Customize the progress bar and seek scrubber appearance using `ScrubberStyle`.

<ResponseField name="height" type="CGFloat" default="3.0">
  Height of the scrubber bar in points.
</ResponseField>

<ResponseField name="knobSize" type="CGFloat" default="12.0">
  Diameter of the draggable scrubber knob in points.
</ResponseField>

<ResponseField name="playedColor" type="Color" default=".white">
  Color of the played (elapsed) portion of the scrubber bar.
</ResponseField>

<ResponseField name="bufferedColor" type="Color" default=".white (50% opacity)">
  Color of the buffered portion of the scrubber bar. Indicates content that has been downloaded but not yet played.
</ResponseField>

<ResponseField name="unplayedColor" type="Color" default=".white (30% opacity)">
  Color of the unplayed (remaining) portion of the scrubber bar.
</ResponseField>

<ResponseField name="position" type="ScrubberPosition" default=".bottom">
  Vertical position of the scrubber relative to the feed view.

  | Value | Description |
  |-------|-------------|
  | `.bottom` | Scrubber sits at the bottom edge of the feed |
  | `.top` | Scrubber sits at the top edge of the feed |
</ResponseField>

<ResponseField name="edgeOffset" type="CGFloat" default="8.0">
  Distance from the view edge in points. The offset is measured from the actual view boundary, not the safe area, so `0` places the scrubber flush against the edge. Useful for positioning the bar right at a tab bar boundary.
</ResponseField>

<ResponseField name="horizontalPadding" type="CGFloat" default="0">
  Horizontal inset from the left and right edges, in points. Set to `0` for a full-width bar.
</ResponseField>

<ResponseField name="cornerRadius" type="CGFloat?" default="nil">
  Corner radius of the scrubber track. When `nil`, defaults to a pill shape (`height / 2`). Set to `0` for sharp edges.
</ResponseField>

<ResponseField name="showKnob" type="Bool" default="false">
  Whether to display the circular drag handle at the current playback position. When `false`, the scrubber renders as a clean progress line without a knob. Users can still tap or drag to seek regardless of this setting.
</ResponseField>

### Scrubber example

A minimal progress bar flush at the bottom edge with no knob (the default):

```swift
let config = FeedConfig(
    theme: ThemeConfig(
        scrubberStyle: ScrubberStyle(
            height: 3.0,
            playedColor: .white,
            bufferedColor: UIColor.white.withAlphaComponent(0.4),
            unplayedColor: UIColor.white.withAlphaComponent(0.2)
        )
    )
)
```

A thicker scrubber with a visible knob, offset from the bottom, and rounded padding:

```swift
let config = FeedConfig(
    theme: ThemeConfig(
        scrubberStyle: ScrubberStyle(
            height: 5.0,
            knobSize: 16.0,
            playedColor: .systemBlue,
            position: .bottom,
            edgeOffset: 12.0,
            horizontalPadding: 16.0,
            cornerRadius: 2.5,
            showKnob: true
        )
    )
)
```

## Caption styling

Customize the appearance of captions and subtitles using `CaptionStyle`.

<ResponseField name="fontSize" type="CGFloat" default="16.0">
  Caption text size in points. Respects the user's Dynamic Type accessibility setting as a scaling baseline.
</ResponseField>

<ResponseField name="textColor" type="Color" default=".white">
  Caption text color.
</ResponseField>

<ResponseField name="backgroundColor" type="Color" default=".black">
  Background color behind caption text.
</ResponseField>

<ResponseField name="backgroundOpacity" type="CGFloat" default="0.7">
  Opacity of the caption background (0.0 to 1.0).
</ResponseField>

<ResponseField name="position" type="CaptionPosition" default=".bottom">
  Vertical position of captions on screen.

  | Value | Description |
  |-------|-------------|
  | `.bottom` | Captions appear near the bottom of the video, above the scrubber |
  | `.top` | Captions appear near the top of the video, below the safe area |
</ResponseField>

### Caption example

```swift
let config = FeedConfig(
    controls: ControlsConfig(
        captions: CaptionConfig(
            enabled: true,
            defaultOn: true,
            style: CaptionStyle(
                fontSize: 18.0,
                textColor: .white,
                backgroundColor: .black,
                backgroundOpacity: 0.8,
                position: .bottom
            )
        )
    )
)
```

## Complete theme example

<Tabs>
  <Tab title="SwiftUI">
    ```swift
    import ShortKit

    let shortKit = ShortKit(
        apiKey: "pk_live_your_publishable_key",
        config: FeedConfig(
            controls: ControlsConfig(
                captions: CaptionConfig(
                    enabled: true,
                    defaultOn: false,
                    style: CaptionStyle(
                        fontSize: 16.0,
                        textColor: .white,
                        backgroundColor: .black,
                        backgroundOpacity: 0.7,
                        position: .bottom
                    )
                ),
                muteOnStart: true,
                share: ShareConfig(enabled: true),
                autoHideControls: AutoHideConfig(
                    enabled: true,
                    delayMs: 3000
                )
            ),
            theme: ThemeConfig(
                primaryColor: .orange,
                backgroundColor: UIColor(red: 0.04, green: 0.04, blue: 0.04, alpha: 1.0),
                controlsTint: .white,
                controlsBackgroundOpacity: 0.5,
                fontFamily: "System",
                scrubberStyle: ScrubberStyle(
                    height: 3.0,
                    playedColor: .orange,
                    bufferedColor: UIColor.white.withAlphaComponent(0.4),
                    unplayedColor: UIColor.white.withAlphaComponent(0.2),
                    position: .bottom,
                    edgeOffset: 0
                ),
                captionStyle: CaptionStyle(
                    fontSize: 16.0,
                    textColor: .white,
                    backgroundColor: .black,
                    backgroundOpacity: 0.75,
                    position: .bottom
                )
            )
        )
    )

    struct ContentView: View {
        var body: some View {
            ShortKitFeedView(shortKit: shortKit)
        }
    }
    ```
