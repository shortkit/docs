The ShortKit Admin Portal is a web dashboard for managing your short-form video feed platform. From the portal you can upload and organize content, configure how your feed behaves, monitor engagement analytics, and manage API keys, webhooks, and team access.

<Note>
  Access the Admin Portal at [portal.shortkit.dev](https://portal.shortkit.dev)
</Note>

## Dashboard overview

When you log in, the Overview dashboard surfaces key metrics at a glance.

### Key metrics

- **Total views** -- video impressions over the selected period
- **Unique viewers** -- distinct users who watched content
- **Total watch time** -- cumulative seconds of video watched
- **Average session duration** -- average time spent in the feed per visit
- **Feed depth** -- average number of videos watched per session
- **Completion rate** -- percentage of videos watched to the end

### Quick actions

- Upload new content
- View recent uploads and processing status
- Check active experiments
- View top-performing content

### Health indicators

- SDK integration status
- API request volume
- Error rates
- Content processing status

## Quick start

### First-time setup

<Steps>
  <Step title="Get your API keys">
    Go to **Settings > API Keys** and copy your publishable key (`pk_live_` or `pk_test_`) for SDK initialization.
  </Step>
  <Step title="Set up webhooks">
    Go to **Settings > Webhooks** to receive notifications when content finishes processing.
  </Step>
  <Step title="Upload content">
    Go to **Content > Upload** and add your first videos via drag-and-drop or URL.
  </Step>
  <Step title="Configure the feed">
    Go to **Feed > Configuration** to set up remote config overlays that control feed behavior.
  </Step>
</Steps>

### Common tasks

| Task | Location |
|------|----------|
| Upload a video | Content > Upload |
| Check processing status | Content > Library (filter by status) |
| Pin content to top of feed | Content > Library > select item > Editorial |
| Schedule content | Content > Library > select item > set publish/expiry dates |
| Adjust feed configuration | Feed > Configuration |
| Create an A/B test | Experiments > Create New |
| View engagement metrics | Analytics > Overview |
| Generate API keys | Settings > API Keys |
| Add team members | Settings > Team |

## Keyboard shortcuts

| Shortcut | Action |
|----------|--------|
| `Cmd+K` / `Ctrl+K` | Open command palette |
| `Cmd+U` / `Ctrl+U` | Quick upload |
| `Cmd+/` / `Ctrl+/` | Show shortcuts |
| `G then C` | Go to Content |
| `G then A` | Go to Analytics |
| `G then S` | Go to Settings |

| Browser | Supported |
|---------|-----------|
| Chrome | 90+ |
| Firefox | 90+ |
| Safari | 14+ |
| Edge | 90+ |

<Tip>
  The portal is responsive and works on tablets for quick tasks like checking analytics. For content upload and detailed configuration, a desktop browser is recommended.
</Tip>
